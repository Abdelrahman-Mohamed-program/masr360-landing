## UI UX Pro Max Skill
This project uses the UI UX Pro Max skill located at /boody/ui-ux-pro-max/

To search the design database:
python3 /boody/ui-ux-pro-max/scripts/search.py "<query>" --domain <domain>

Available domains: style, color, typography, landing, chart, ux, product

Use this skill before making ANY design decision.
